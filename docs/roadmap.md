# Roadmap

A high-level look at what's planned for ESPHome Media Player. This is a living document — features may shift in priority as the project evolves.

Have an idea? [Open a feature request](https://github.com/jtenniswood/esphome-media-player/issues/new) on GitHub.

## Planned

| Feature | Description |
| --- | --- |
| Media Browsing | Browse and select media directly from the display — playlists, albums, etc |
| Lighting Control | Overlay control for room lights and scenes |
