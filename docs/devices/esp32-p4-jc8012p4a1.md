# Guition ESP32-P4 JC8012P4A1 (10.1")

10.1" (1280×800) touchscreen panel with ESP32-P4 and ESP32-C6 coprocessor for WiFi/Bluetooth. Uses a side-panel layout: album art on the left, track info on the right. Tested with Google and Sonos speakers via Home Assistant.

![Example 1](../images/guition-esp32-p4-jc8012p4a1-example1.jpg)

## Install

Click **Install** to flash the firmware directly from your browser. No ESPHome dashboard or YAML required.

<InstallButton device="jc8012p4a1" />

::: tip
If you prefer to install via the ESPHome dashboard, see [ESPHome Config](/advanced/esphome-config) (this device uses the `guition-esp32-p4-jc8012p4a1` package).
:::
